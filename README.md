### 조건문 if 문에 대해 알아봅니다

```python
a = 33
b = 200

if b > a:
  print("b is greater than a")

```

```python
a = 33
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
```



```python
a = 200

  print("a and b are equal")

```
